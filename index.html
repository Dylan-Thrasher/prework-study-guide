HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • Alt text can be added to images to help with accessibility.
  • Meta data can alter how the page is read, including image and text scaling
  • Charset/Character Set is important to place for html code to let us use the most characters and symbols.
  • Declarations like !DOCTYPE are not case sensitive. Doctype also tells the browser what document type to expect
  • HTML attributes are always defined in the beginning element.
  • Title element is required for a webpage and extremely important for SEO.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: adds modifications in the current branch to the staging area. -A added on indicates add all changes.
  • git commit -m "message here in quotes": sends the commit along with a message explaining the changes
  • git pull origin main: checks to see if our local branch is in sync with GitHub base branch
  • git push origin branch-name: pushes the changes from our local branch to the remote branch in GitHub.
